### 开发流程
#### 从dev分支拉去代码至本地，创建自己本地分支，在本地分支进行开发，最后合并至dev分支

### 启动项目
#### npm install 
#### npm run build
#### node ./src/index.js
##### 浏览器打开 localhost:3000
